
# Personal Portfolio Website

A responsive personal portfolio website for showcasing my background, technical skills, projects, and contact information.

## Overview

This portfolio is a single-page website built with HTML, CSS, and JavaScript. It includes a fixed navigation bar, smooth scrolling sections, project cards, modal-style project details, and a contact section.

## Features

- Responsive single-page portfolio layout

- Fixed navigation bar with smooth scrolling

- Hero section with call-to-action buttons

- About section for education, experience, and background

- Project cards for highlighting technical work

- Interactive project details

- Contact section for professional outreach

- Custom dark theme with orange accent styling

## Tech Stack

- HTML

- CSS

- JavaScript

- EmailJS

- Git and GitHub

## Project Structure

portfolio/

└── index.html

## Run Locally

Clone the repository:

git clone git@github.com:kp5036/portfolio.git

cd portfolio

Open the site:

open index.html

## Purpose

This project was built to create a centralized personal website for presenting my resume, projects, GitHub, LinkedIn, and contact information.
